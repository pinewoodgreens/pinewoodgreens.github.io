<objective>
Close two verified CSS gaps in css/pwg.css that are blocking A11Y-01 and A11Y-02:

1. `--pine-main` and `--pine-dark` are used for focus ring and skip-link background but are
   never declared in a `:root` block — both properties resolve to `initial` (invalid), making
   the focus ring invisible and the skip link unstyled.

2. `.navbar-nav .open ul { display: none }` inside `@media (min-width: 768px)` overrides
   Bootstrap's `.open > .dropdown-menu { display: block }` with higher specificity, silently
   hiding keyboard-activated dropdown menus on desktop viewports.

Purpose: After this plan, all five Phase 2 WCAG success criteria are structurally satisfied.
Output: Updated css/pwg.css with :root token block inserted and suppression rule removed.
</objective>
